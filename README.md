## 📌 Project Overview
**Festive Gadget Gains** is a comprehensive data analysis and visualization project developed to decode gadget sales trends during major Indian festive seasons. By leveraging **Microsoft Power BI** and **SQL Server Management Studio (SSMS)**, this project analyzes customer demand, product performance, and the impact of promotional offers to support data-driven decision-making for e-commerce businesses.

## 🎯 Key Objectives
* Evaluate overall business performance and total revenue generated during peak festival sales.
* Identify high-demand gadgets and top-performing regional markets.
* Measure the impact of festive discounts and promotional campaigns on sales volume.
* Track logistics efficiency, delivery times, and product return/cancellation trends.

## 🛠️ Tech Stack & Tools
* **Business Intelligence:** Power BI Desktop & Power BI Service
* **Database Management:** Microsoft SQL Server, SQL Server Management Studio (SSMS)
* **Data Processing:** Power Query, Data Analysis Expressions (DAX)

## 🗄️ Database & SQL Integration
The raw data is securely managed within **SQL Server**. Data preprocessing, including handling missing values and formatting, was executed before establishing a robust relational data model connecting *Sales, Products, Customers,* and *Regions*. 

*The project utilizes advanced SQL queries for metric extraction, alongside DAX for calculated columns and measures in Power BI.*

## 📂 Dataset
The analysis is based on the `Indian_Festival_Sales_Enriched_Final.csv` dataset, which contains detailed transaction records including order dates, product categories, pricing, discounts, and regional demographics.

## 🚀 How to Use This Repository
1. Clone the repository to your local machine.
2. Ensure you have **Microsoft Power BI Desktop** installed.
3. Open the `Festive Gadget Gain.pbix` file to interact with the dashboards.
4. (Optional) Review the PDF documentations (`Festive Gadget Gain Documentation.pdf` and `Festive Gadget Gain PPT.pdf`) for an in-depth system analysis and methodology.
